QueryBuilder

  • customSQL(_:)

    Adds a SQL-transforming closure to the QueryBuilder that will be called before the query is executed.

    One or more SQL-transforming closures may be added–they will be run sequentially.

    Declaration

    Swift

    public func customSQL(_ closure: @escaping (inout DataQuery) -> ()) -> Self

    Parameters

    closure

    Accepts the SQL.DataQuery generated by this query builder. Called before the query executes. The SQL.DataQuery is mutable. Add or remove any custom options.
