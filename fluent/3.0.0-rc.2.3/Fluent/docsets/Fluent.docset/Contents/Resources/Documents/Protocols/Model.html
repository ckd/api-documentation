Model

public protocol Model: AnyModel, Reflectable

Fluent database models. These types can be fetched from a database connection using a query.

Types conforming to this protocol provide the basis fetching and saving data to/from Fluent.

  • siblings(related:through:_:_:) Extension method

    Create a siblings relation for this model.

    Unless you are doing custom keys, you should not need to pass any parameters to this function.

    class Toy: Model {
    var pets: Siblings<Toy, Pet, PetToyPivot> {
        return siblings()
    }
}

    See Siblings class documentation for more information about the many parameters. They can be confusing at first!

    Note: From is assumed to be the model you are calling this method on.
