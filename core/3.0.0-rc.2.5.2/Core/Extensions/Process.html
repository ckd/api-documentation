Process

  • execute(_:_:)

    Executes the supplied program in a new process, blocking until the process completes. Any data piped to stdout during the process will be returned as a string. If the process exits with a non-zero status code, an error will be thrown containing the contents of stderr and stdout.

    let result = try Process.execute("echo", "hi")
print(result) /// "hi"
